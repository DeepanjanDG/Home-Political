Screen Reader Access

This page provides information and shortcuts for screen reader users to navigate the website efficiently.

    The website complies with World Wide Web Consortium (W3C) Web Content Accessibility Guidelines (WCAG) 2.0 level AA. This will enable people with visual impairments to access the website using assistive technologies, such as screen readers. The information on the website is accessible with different screen readers.

  • Press Alt + 1 to go to the Home page.
  • Press Alt + 2 to skip to the main content.
  • Press Alt + 3 to access the Contact Us page.
  • Press Alt + 4 to view the Sitemap.
